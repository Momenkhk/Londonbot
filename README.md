# Londonbot — Ticket System (Components v2 style)

تم تحديث النظام حسب الملاحظة:
- يوجد الآن **أمرين** رسميًا:
  - `/ticket-panel`
  - `/ticket-setup`
- تم نقل الإعدادات إلى `config.json` بدل `.env`.

## الأوامر

1. `/ticket-panel`
   - يعرض واجهة النظام ويؤكد وجود الأمر الثاني.
   - يحتوي زر **بدء الإعداد**.

2. `/ticket-setup`
   - يبدأ إعداد بانل التذاكر مباشرة (مودال + سيليكت منيو + أزرار).

## الميزات

- إعداد عنوان/رسالة/صورة الامبد عبر Modal.
- اختيار نوع التذاكر (Menu أو Buttons).
- اختيار عدد التذاكر (1–10).
- اختيار اللون.
- اختيار صورة الجانب (صورتك/السيرفر/البوت/لا شيء).
- اختيار روم نشر البانل.
- إنشاء قناة تذكرة تلقائيًا عند الضغط.
- أزرار داخل التذكرة: استلام + إغلاق.

## الإعداد

عدّل `config.json`:

```json
{
  "token": "YOUR_BOT_TOKEN",
  "guildId": "YOUR_GUILD_ID",
  "staffRoleId": "YOUR_STAFF_ROLE_ID"
}
```

> إذا لا تريد قيمة، اتركها كـ string فارغ `""`.

## التشغيل

```bash
npm install
npm start
```
